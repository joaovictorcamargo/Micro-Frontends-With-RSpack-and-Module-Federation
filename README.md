# Micro-Frontends With RSpack and Module Federation



### Cloning the repository

```shell
git clone https://github.com/joaovictorcamargo/Micro-Frontends-With-RSpack-and-Module-Federation.git
```

### Install packages

```shell
pnpm i
```

### Start the app

```shell
pnpm start
```
